Template Literals

Template Literals

Los template literals son cadenas de texto que permiten expresiones incrustadas y pueden abarcar múltiples líneas. Se definen usando backticks (`) en lugar de comillas simples o dobles.

Sintaxis Básica

const nombre = 'Humberto';
const edad = 33;

// Concatenación tradicional
const mensaje1 = 'Hola, soy ' + nombre + ' y tengo ' + edad + ' años';

// Template literal
const mensaje2 = `Hola, soy ${nombre} y tengo ${edad} años`;

Ventajas:

  • Sintaxis más limpia y legible
  • Interpolación de variables con ${}
  • Soporte para múltiples líneas
  • Puede incluir expresiones

Múltiples Líneas

const html = `

${nombre}

Edad: ${edad}

`;

const mensaje = ` Primera línea Segunda línea Tercera línea `;

Características:

  • Mantiene el formato y los saltos de línea
  • Ideal para crear HTML
  • No necesita caracteres de escape
  • Respeta la indentación

Expresiones y Cálculos

const a = 5;
const b = 10;

// Cálculos
const suma = `${a} + ${b} = ${a + b}`;

// Operador ternario
const estado = `${a > b ? 'Mayor' : 'Menor'}`;

// Llamadas a funciones
const mayusculas = `${nombre.toUpperCase()}`;

Posibilidades:

  • Operaciones matemáticas
  • Operadores ternarios
  • Llamadas a métodos
  • Cualquier expresión JavaScript válida

Casos de Uso Comunes

Templates HTML

const crearCard = (usuario) => `

${usuario.nombre}

${usuario.descripcion}

`;

Mensajes Dinámicos

const mensaje = (tipo, contenido) => ` [${tipo.toUpperCase()}]: ${contenido} Fecha: ${new Date().toLocaleString()} `;
