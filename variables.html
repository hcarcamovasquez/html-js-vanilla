Variables en JavaScript

Introducción a las Variables

En JavaScript moderno, tenemos tres formas de declarar variables: var, let y const. Cada una tiene sus propias características y casos de uso específicos.

var

Características:

  • Tiene alcance de función o global
  • Puede ser redeclarada
  • Puede ser reasignada
var nombre = 'Juan';
var nombre = 'Pedro'; // Permitido
nombre = 'María'; // Permitido

function ejemploVar() {
var x = 1;
if (true) {
var x = 2;
console.log(x); // 2
}
console.log(x); // 2
}

let

Características:

  • Tiene alcance de bloque
  • No puede ser redeclarada en el mismo ámbito
  • Puede ser reasignada
let edad = 25;
// let edad = 26; // Error!
edad = 26; // Permitido

function ejemploLet() {
let x = 1;
if (true) {
let x = 2; // Variable diferente
console.log(x); // 2
}
console.log(x); // 1
}

const

Características:

  • Tiene alcance de bloque
  • No puede ser redeclarada
  • No puede ser reasignada
  • Los objetos y arrays pueden ser modificados [IMPORTANTE]
const PI = 3.14159;
// PI = 3.14; // Error!

const persona = {
nombre: 'Ana'
};
persona.nombre = 'María'; // Permitido
// persona = {}; // Error!

Mejores Prácticas

✅ Recomendado:

  • Usar const por defecto
  • Usar let cuando necesites reasignar valores
  • Evitar var en código moderno

❌ Evitar:

  • Usar var para nuevos proyectos
  • Reasignar variables innecesariamente
  • Declarar variables sin inicializarlas
