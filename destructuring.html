Destructuración en JavaScript

Destructuración (Destructuring)

La destructuración permite extraer valores de objetos o arrays y asignarlos a variables de una manera más concisa y legible.

Destructuración de Objetos

const carro = {
marca: 'Toyota',
modelo: 'Corolla',
año: 2023
};

// Destructuración básica
const { marca, modelo } = carro;

// Con alias
const { marca: fabricante, modelo: tipo } = carro;

// Valores por defecto
const { color = 'negro' } = carro;

Características:

  • Extrae propiedades por nombre
  • Permite asignar alias a las propiedades
  • Puede tener valores por defecto
  • Funciona con objetos anidados

Destructuración de Arrays

const numeros = [1, 2, 3, 4, 5];

// Destructuración básica
const [primero, segundo] = numeros;

// Saltando elementos
const [, , tercero] = numeros;

// Rest operator
const [inicio, ...resto] = numeros;

Características:

  • Extrae elementos por posición
  • Permite saltar elementos usando comas
  • Usa rest operator (...) para el resto de elementos
  • Puede tener valores por defecto

Operador Spread (...)

// Copiar arrays
const original = [1, 2, 3];
const copia = [...original];

// Combinar arrays
const combinado = [...original, 4, 5];

// Copiar objetos
const persona = {
nombre: 'Ana',
edad: 25
};
const copiaPerson = { ...persona };

Usos comunes:

  • Crear copias de arrays y objetos
  • Combinar múltiples arrays
  • Añadir elementos a un array
  • Combinar objetos

Casos Prácticos Básicos

Parámetros de Función

function procesarUsuario({ nombre, edad, ciudad = 'Desconocida' }) {
return `${nombre}, ${edad} años, ${ciudad}`;
}

Intercambio de Variables

let a = 5, b = 10;
[a, b] = [b, a];

Casos Prácticos Avanzados

3. Configuración de Componentes

function createButton({
text = 'Click',
style = {},
onClick = () => {},
disabled = false,
...otherProps
} = {}) {
return {
text,
style,
onClick,

...otherProps

}

4. Manipulación de Arrays y Objetos

// Combinando arrays y objetos
const config = {
opciones: ['a', 'b'],
valores: { x: 1, y: 2 }
};

const {
opciones: [primerOpcion, segundaOpcion],
valores: { x, y }
} = config;

5. Funciones con Destructuración Múltiple

function procesarDatos([primer, segundo, ...resto] = [], { orden = 'asc', filtro = null } = {}) {
// Procesar array
const datos = orden === 'asc'
? [primer, segundo, ...resto]
: [...resto, segundo, primer];

// Aplicar filtro si existe
return filtro ? datos.filter(filtro) : datos;
}

// Ejemplo de uso
const resultado = procesarDatos([1, 2, 3, 4], {
orden: 'desc',
filtro: x => x > 2
});

Tips y Buenas Prácticas

  • Usa valores por defecto para evitar errores cuando faltan propiedades
  • Combina destructuración con el operador spread para mayor flexibilidad
  • En funciones, usa la destructuración de parámetros para código más limpio
  • Aprovecha los alias cuando necesites evitar conflictos de nombres
  • Utiliza la destructuración anidada con moderación para mantener la legibilidad
